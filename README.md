# 🌊 Экзаменационный проект: «Цифровой Океанариум»

Репозиторий содержит исходный код интерактивной веб-страницы «Океанариум», разработанной в рамках практического экзаменационного задания.

---

## 📋 Статус выполнения ТЗ

- [x] **Шаг 1:** Разработка структуры и адаптивная верстка HTML-страницы на тему «Океанариум».
- [x] **Шаг 2:** Подготовка репозитория на GitHub, выгрузка исходного кода и публикация через хостинг **GitHub Pages**.
- [x] **Шаг 3:** Интеграция трехмерной модели «Морская черепаха» (формат `.glb`) в тело страницы с поддержкой интерактивного управления.

---

## 🛠 Использованный стек технологий
* **HTML5** — семантическая структура веб-документа.
* **CSS3** — стилизация интерфейса и адаптивная сетка (CSS Grid / Flexbox).
* **Google <model-viewer>** — библиотека для рендеринга интерактивной 3D-графики в веб-среде с поддержкой аппаратного ускорения.
* **GitHub Pages** — хостинг для публикации проекта в сети.
